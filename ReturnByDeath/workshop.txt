title=Return by Death (Re:Zero)
description=Death is not the end - it is a checkpoint.
description=
description=This mod recreates the Return by Death mechanic from Re:Zero - Starting Life in Another World as a Project Zomboid negative trait (+1 point):
description=
description=- Your loop silently anchors a safe point on an interval, or set one manually via right-click.
description=- When you die, you seamlessly return to your safe point with your inventory restored - no corpse, no death screen.
description=- Each return wounds your mind: depression, stress and panic stack up, and escalate the more you loop.
description=- The taboo is real: right-click another player and "Tell them about Return by Death..." - the Witch crushes their heart (server-validated, can be disabled in Sandbox settings).
description=- Optional Witch's Scent: the miasma of a fresh return briefly attracts the dead.
description=- Everything is tunable on the Return by Death Sandbox page: checkpoint interval, penalty amounts and escalation, return limits per day, health restore, guard threshold, and more.
description=
description=Fully multiplayer compatible: returns are per-player, the taboo kill is server-authoritative, and sandbox options are read from the server.
description=
description=Note on how the loop works: Return by Death fires at the brink of death (a configurable health threshold), so the base-game death pipeline - corpse, death screen - never runs. The world itself is not rewound (loot, zombies and map changes persist); the reset is player-scoped.
